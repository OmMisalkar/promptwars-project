# Neon Aura AR – Hand Tracking Experience 🚀

This project is an advanced **Augmented Reality (AR) web application** built using **MediaPipe Hands**.

## 🔥 Features

* Real-time hand tracking using webcam
* Gesture detection (pinch, open hand, etc.)
* Interactive particle effects and neon visuals
* Multiple themes (Cyberpunk, Galaxy, Ocean, etc.)
* Audio-reactive effects
* Smooth animations with canvas rendering

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* MediaPipe Hands
* Web Audio API
* Canvas API

## 🎯 How to Run

1. Open `index.html` in browser
2. Allow camera permission
3. Click "Enter Experience"
4. Enjoy AR interaction ✨
